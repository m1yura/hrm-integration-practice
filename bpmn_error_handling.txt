Легенда:
- Ромбы - точки принятия решений (ошибка?)
- Прямоугольники - действия (Retry, Circuit Breaker, Compensation)
- Круги - события (Start, End)
- Жирные линии - поток компенсации

Типы ошибок и обработка:
1. Network Error → Retry (3 попытки, экспоненциальная задержка)
2. Service Unavailable → Circuit Breaker → OPEN state (30 сек)
3. Business Logic Error → Compensation Saga (откат операций)
4. Validation Error → Log + Notification (без ретрая)
